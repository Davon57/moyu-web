# 摸鱼网站 - 产品设计文档

## 1. 项目概述

- **项目名称**: 摸鱼网站 (MoYu)
- **项目类型**: 纯前端 Web 应用 (SPA)
- **核心定位**: 聚合各类摸鱼内容，让打工人在工作间隙轻松一下
- **slogan**: "偷偷摸鱼，快乐无边" 🐟

---

## 2. 用户画像

### 目标用户

| 特征 | 描述 |
|-----|------|
| 年龄 | 22-35 岁 |
| 职业 | 上班族、学生党 |
| 使用场景 | 上班/上课间隙、午休、蹲坑时 |
| 使用目的 | 短暂放松、消磨时间 |
| 设备偏好 | 手机为主 (70%)，电脑 (20%)，平板 (10%) |

### 用户需求

1. **快速获取信息** - 不需要深度阅读，扫一眼就能获取有趣内容
2. **轻量互动** - 点赞、刷新、分享等简单操作
3. **随时可停** - 随时退出回到工作状态
4. **个性化** - 喜欢能定制自己的内容偏好

---

## 3. 功能模块详情

### 3.1 第一期功能 (MVP)

#### 📰 今日新闻
- **功能描述**: 聚合各大平台热搜榜
- **数据来源**: 微博热搜、知乎热榜、抖音热榜、B站热门等
- **交互**:
  - 展示标题 + 热度数值
  - 点击跳转原平台
  - 每5分钟自动刷新
  - 可切换不同平台

#### 😂 摸鱼笑话
- **功能描述**: 随机推送搞笑段子
- **数据来源**: 你提供的 API
- **交互**:
  - 每次展示1条笑话
  - "换一条" 按钮刷新
  - 点赞/不喜欢 反馈
  - 底部展示热门标签

#### 🔥 热榜聚合
- **功能描述**: 同时展示多个平台的热搜
- **布局**: 标签页切换不同平台
- **数据来源**: 你提供的 API

### 3.2 第二期功能

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 🌵 毒鸡汤 | 每日一句扎心文案 | P1 |
| ⏰ 下班倒计时 | 设置下班时间，实时倒计时 | P1 |
| 🌤️ 天气查询 | 查询当前位置天气 | P2 |
| 📅 历史上的今天 | 历史上的今天发生了什么 | P2 |
| 🎮 摸鱼小游戏 | 简易小游戏入口 | P3 |

## 5. UI/UX 设计

### 5.1 响应式断点

| 设备 | 断点 | 布局 |
|-----|------|------|
| 手机 | < 768px | 单列，卡片垂直排列 |
| 平板 | 768px - 1024px | 2列网格 |
| 电脑 | > 1024px | 3列网格 |

### 5.2 配色方案

**明亮模式**
- 主色: `#3B82F6` (蓝色)
- 背景: `#FFFFFF`
- 次背景: `#F9FAFB`
- 文字: `#1F2937`
- 边框: `#E5E7EB`

**暗色模式**
- 主色: `#60A5FA` (浅蓝)
- 背景: `#1F2937`
- 次背景: `#111827`
- 文字: `#F9FAFB`
- 边框: `#374151`

### 5.3 组件设计

| 组件 | 规格 |
|-----|------|
| 卡片 | 圆角 12px，阴影，轻微 hover 上浮 |
| 按钮 | 圆角 8px，主色填充 |
| 标签 | 圆角 16px，药丸形状 |
| 加载 | 骨架屏 + 旋转图标 |

### 5.4 动效

- 页面切换: fade 过渡 200ms
- 卡片 hover: transform translateY(-4px)
- 按钮点击: scale(0.98)
- 刷新: 旋转图标
