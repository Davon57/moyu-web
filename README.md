# 摸鱼网站 🐟

> 偷偷摸鱼，快乐无边

一个聚合各类摸鱼内容的纯前端网站，让打工人在工作间隙轻松一下。

## 功能

- 📰 今日新闻 - 各大平台热搜榜汇总
- 😂 摸鱼笑话 - 随机推送搞笑段子
- 🔥 热榜聚合 - 微博、知乎、抖音热搜
- 🌵 毒鸡汤 - 每日一句扎心文案
- ⏰ 下班倒计时 - 距离下班还有...
- 🌤️ 天气查询 - 查看今日天气

## 技术栈

- Vue 3 (Composition API)
- Vite
- Vue Router 4
- Pinia
- Axios

## 运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 响应式

- 📱 手机端
- 📟 平板端
- 💻 电脑端
